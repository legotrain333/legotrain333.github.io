Software I Use

OnShape

Onshape is a cloud-based 3D CAD platform that allows users to create precise, parametric models directly in a web browser. It supports real-time collaboration, version control, and works on any device without the need for installation. Onshape is ideal for designing parts for 3D printing or CNC machining and can export files in formats like STL, STEP, and DXF for use with slicers or CAM software.

OrcaSlicer

OrcaSlicer is a powerful and user-friendly slicing software designed for FDM 3D printers. Based on Bambu Studio and originally derived from PrusaSlicer, it supports a wide range of printers and advanced features like pressure advance, input shaping, and variable layer heights. OrcaSlicer is ideal for preparing models for high-quality 3D prints, including multi-material and color printing with AMS systems.

Easel

Easel is an intuitive, web-based CNC design and control platform developed by Inventables. It combines CAD and CAM in a single interface, making it easy to design, simulate, and send toolpaths to desktop CNC machines. Easel is compatible with machines like the Genmitsu 3018 series and supports engraving and milling workflows with tools for material selection, bit setup, and depth control.
